# Flappy Bird

A simple Flappy Bird clone built with Python and Tkinter. Control the bird, avoid obstacles, and try to achieve the highest score possible.

## Features

- Bird movement with gravity physics
- Obstacle generation
- Collision detection
- Score tracking
- Game Over screen
- Simple graphical user interface built with Tkinter

## Getting Started

### Requirements

- Python 3.10 or newer

### Installation

Clone the repository:

```bash
git clone https://github.com/V-Lukac/Flappy-Bird.git
cd Flappy-Bird
